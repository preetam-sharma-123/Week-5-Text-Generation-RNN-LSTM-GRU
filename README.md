# Text Generation using Vanilla RNN, LSTM and GRU

## Overview

This project compares three recurrent neural network architectures for next-word prediction and text generation:

* Vanilla RNN (SimpleRNN)
* Long Short-Term Memory (LSTM)
* Gated Recurrent Unit (GRU)

The models are trained on a custom text corpus using TensorFlow/Keras.

## Objectives

* Tokenize text corpus
* Create n-gram sequences
* Train SimpleRNN, LSTM, and GRU models
* Compare training loss
* Generate text from a seed phrase
* Analyze performance differences

## Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* Pandas

## Beginner Enhancements

* Custom text corpus
* Embedding dimension increased
* Training extended to 200 epochs
* Hidden layer size increased
* Generated text expanded to 10 words

## Results

The notebook includes:

* Loss comparison graph
* Generated text samples
* Model performance comparison
